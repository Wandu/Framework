Wandu Caster
===

Perfect Type Casting Library.

## Usage

```php
$caster = new \Wandu\Caster\Caster;

$caster->cast('10,20,30,40', 'int[]'); // return [10, 20, 30, 40];
```

## Support Cast Types

**Plain**

- `string`
- `int`
- `integer`
- `num`
- `number`
- `float`
- `double`
- `bool`
- `boolean`

**Arrayble**

- `string[]`
- `int[]`
- `integer[]`
- `num[]`
- `number[]`
- `float[]`
- `double[]`
- `bool[]`
- `boolean[]`

**Nullable**

- `string?`
- `int?`
- `integer?`
- `num?`
- `number?`
- `float?`
- `double?`
- `bool?`
- `boolean?`
